# Retail Sales Excel Dashboard

An Excel dashboard built on US retail sales data covering 2014–2017. Tracks sales, profit, and customer trends across categories and states.

## What's in it

- **Dashboard** — Visual overview of all key metrics
- **Sales vs Category** — Top selling product sub-categories
- **Category vs Sales** — Yearly profit by category (Furniture, Office Supplies, Technology)
- **Month vs Sales** — Monthly sales trend across the year
- **Top 5 Customers** — Highest profit-generating customers
- **State** — Sales breakdown by US state
- **Customer vs Year** — Customer count per year

## The dashboard helps track:

- Total sales
- Profit / revenue trends
- Category performance
- Region or market comparison
- Monthly trends

## Tools Used

- Microsoft Excel
- Pivot Tables
- Charts & Slicers
- Dashboard design

## Data Overview

| Column | Description |
|--------|-------------|
| Order Date | Date of purchase |
| Customer Name | Buyer name |
| State | US state of order |
| Category / Sub-Category | Product type |
| Sales | Revenue per order |
| Quantity | Units ordered |
| Profit | Profit per order |
| Year / Month | Time breakdown |

## Key Insights

- California is the #1 state by sales
- Phones and Chairs are the top sub-categories
- Technology brings the most profit year over year
- Tamara Chand is the highest profit customer

## Recommendations

- Focus on strong-performing categories
- Improve low-performing regions
- Monitor monthly sales trends regularly

## How to use

Download the `.xlsx` file and open in Excel (2016 or later). Use slicers on the dashboard to filter by year or category.
